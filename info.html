Minesweeper

Minesweeper is a strategy game created in the 60's, which consists of exploring a minefield without detonating any bomb, for this, next to each bomb there are numbers, which represent the number of bombs around that cell, for this task, the player has a limited number of flags to which the mined cells must be assigned.

The first Minesweeper, made by Jerimac Ratliff's (Wikipédia)

This game was made by Gabriel Cambraia, Tomás Carvalho & Daniel Ferra, usinge web technologies such as HTML, CSS and Javascript.
